solusi untuk menengahkan laman:

body {
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
}

justify-content: center; untuk menengahkan secara horizontal.

align-items: center; untuk menengahkan secara vertikal.
