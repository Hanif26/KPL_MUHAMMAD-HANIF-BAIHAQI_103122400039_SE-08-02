program awal menghasilkan output 0 dikarenakan >= yang artinya lebih besar sama dengan, dengan ini 0 diikut sertakan yang mana angka manapun jika di kalikan 0 hasilnya akan tetap 0

```
        if (arr[i] >= 0) {
                result = result * arr[i];
                }
```

maka disini di perbaiki menjadi
```
        if (arr[i] > 0) {
                result = result * arr[i];
                }
```
kesimpulan nya
pada function awal yang digunakan sum, maka kita mengubah ke MulOfArray(arr). pada algoritma awal terdeteksi >= yang mana ini mengikutsertakan 0, maka penyelesainya yaitu menghapus "=" agar 0 tidak tesortir.
